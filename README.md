# CSI-M1 - Gestion des Stages IDMC

Application web de gestion des stages, alternances et CDD pour les étudiants de l'IDMC.

## Stack Technique

- **Backend**: Node.js + Express
- **Frontend**: HTML/CSS/JavaScript
- **Base de données**: PostgreSQL

## Structure du Projet

```
├── backend/          # API Node.js
├── frontend/         # Interface web
├── database/         # Scripts SQL
└── README.md
```

## Installation

```bash
# Cloner le repo
git clone https://github.com/abdellahi54/CSI-M1.git

# Backend
cd backend
npm install
npm run dev

# La BD doit être configurée sur le serveur de l'université
```
